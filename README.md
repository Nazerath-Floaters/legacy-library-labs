# Legacy Library Labs

A modern landing site prototype for a side business focused on:
- media archival and preservation
- curated retro playback systems
- Batocera-based console and device delivery
- premium presentation around ownership, nostalgia, and long-term access

## Stack
- React
- Vite

## Local development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Current homepage sections
- Hero / premium landing section
- Archive media showcase
- Get more from your media
- Pick your console
- Legacy archive callout

## Notes
This is the first branded site scaffold for Legacy Library Labs.
